# Product Context

## Why This Project Exists

This repository is being executed through Nautex task orchestration.
The current objective is to establish the project foundation before feature work.

## Problem It Solves

- Provides an implementation workspace where tasks are delivered and tracked by Nautex.
- Ensures foundation and safety milestones are completed in a structured order.

## How It Should Work

- Agent fetches focus tasks from Nautex.
- Agent executes only in-scope files and requirements.
- Agent updates task statuses (`In progress` -> `Done`) via Nautex.

## Current UX / Delivery Goals

- Reliable environment bootstrap through Docker-based infrastructure.
- Clear, repeatable local development flow.
- Safe progression through milestone tasks with full traceability.
